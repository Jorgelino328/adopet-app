# Pet shop

Aplicativo de petshop feito com Flutter.

## Como executar

1. Entre na pasta do projeto:
   ```bash
   cd /home/jorgelino/pet-shop
   ```

2. Habilite o suporte para web, se ainda não estiver ativado:
   ```bash
   flutter config --enable-web
   ```

3. Execute o app em modo desenvolvimento:
   ```bash
   flutter run -d web-server --web-port=3000
   ```

4. Abra no navegador o endereço:
   ```text
   http://localhost:3000
   ```

## Testar a build web

```bash
flutter build web
```

Os arquivos prontos para publicação ficam em `build/web`.
